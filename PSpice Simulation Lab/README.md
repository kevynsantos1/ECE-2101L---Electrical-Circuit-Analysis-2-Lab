In this lab we will be using PSpice, a circuit simulation tool, to analyze and verify different types
of circuit setups. We will be simulating the following scenarios: Transient Analysis of a
Switched RC Circuit, Transient Analysis of a Switched RLC Circuit, Transient Analysis of a
Circuit with an AC Voltage Source, Transient Analysis of an RC Circuit Switch, Steady-State
AC Analysis of an RC Circuit, AC Sweep of an RC Circuit, Using Plot Window Templates -
Gain and Phase, Noise Analysis - Plotting Voltage Spectral Density, Second-Order Low-Pass
Filter Gain, and Parametric Sweep. All of these will help us grow our familiarity with how to use
PSpice and how to effectively analyze different circuits.
