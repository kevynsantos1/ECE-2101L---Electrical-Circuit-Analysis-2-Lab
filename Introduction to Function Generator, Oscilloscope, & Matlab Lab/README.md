In this lab, we will explore the ways in which we use function generators and oscilloscopes to
create, analyze, measure, and visualize many aspects of a sinusoidal voltage waveform.
Function generators as the name suggests can create various voltage functions that are useful
in powering circuits using sine wave voltage and in digital logic as a simple clock cycle using
square waves. Oscilloscopes on the other hand are used to visualize and measure these
moving voltages that are not possible to analyze using traditional multimeters.
